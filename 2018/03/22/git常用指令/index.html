四、操作用遇到的问题

提交到远端并不是所有人都可以提交的，所以输入上述命令的过程中，输入第6行命令后会报错，因为我们本地并未生成SSH Keys。此时我们需要打开登录状态下的github官网，点击个人头像->Settings->SSH and GPG keys->New SSH Keys，如何新SSH Keys请参考本文第一条。
