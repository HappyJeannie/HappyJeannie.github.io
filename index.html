CSS入门

CSS全称为层叠样式表(Cascading Style Sheets，简写CSS)。

一、CSS常用网站推荐

  • CSS揭秘
  • CSS 2.1 中文 spec
  • Magic of CSS 免费在线书
二、如何引入CSS

1、行内样式 优先级最高
2、内联样式 优先级仅次于行内
3、外部样式 优先级最低
4、@import url() 样式表中引入其他样式表 优先级同上

三、布局与定位

1）div的高度是由其内部文档流元素高度总和决定的
2）文档流：文档内元素的流动方向，流动方向为从左往右，从上往下
3）font-size:100px，这个100px指的是这个字体里面最高的位置到最低的位置的距离，并且每种字体的建议行高都不一样，为了使文档元素不同字体相同文字的高度一致，可以对其设置line-height来控制其高度。但是在较大字号的时候高度还是会有不一致的情况。
4)在写css样式的时候尽量不要去定义元素的宽高，最好是由内向外写，先写padding，然后是border，最后是margin，这样写的话如果文本文字扩展的话不会导致页面文字被隐藏。
5）通过控制border来绘制一个三角形
